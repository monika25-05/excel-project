# 📊 Excel Project - Electronic Shop Sales Analysis

This project focuses on analyzing electronic shop sales data using Excel.  
The goal is to identify sales patterns, customer preferences, and trends that help improve business decision-making.

---

## 📁 Dataset Used

📄 File Name: `Electronic Shop Dataset_new project.xlsx`

The dataset includes the following columns:

- Customer name & gender  
- Product category  
- Purchase amount  
- Location  
- Date of purchase  
- Quantity & profit details
